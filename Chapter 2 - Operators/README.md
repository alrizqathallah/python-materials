# Operators

Operator adalah fungsi yang digunakan untuk mengoperasikan variabel dan nilai dengan operasi tertentu.

Beberapa jenis operator yang disediakan oleh Python:

- Arithmetic Operators
- Assignment Operators
- Comparasion Operators
- Logical Operators
- Identity Operators
- Membership Operators
- Bitwise Operators
- Operator Concatenation

## Arithmetic Operators

Operator Aritmatika adalah simbol-maupun karakter yang digunakan untuk melakukan operasi matematika standar. Seperti penjumlahan, pengurangan, perkalian, pembagian dsb.

1. Penjumlahan
Menambahkan dua nilai atau lebih.

```Python
# Penjumlahan

y = 20

print(10 + 20)
```

2. Pengurangan
Mengurangi nilai di sisi kiri dengan nilai di sisi kanan.
```Python
# Pengurangan
x = 10
y = 5
z = x - y
print(x - y)
print(z)
print(10 - 5)
```

3. Perkalian
Mengalikan dua nilai.
```Python
# Perkalian
x = 10
y = 2
z = x * y
print(x * y)
print(z)
print(10 * 2)
```

4. Pembagian
Membagi nilai di sisi kiri dengan nilai di sisi kanan. Hasilnya selalu berupa bilangan desimal (float).
```Python
# Pembagian
x = 10
y = 5
z = x / y
print(x / y)
print(z)
print(10 / 5)
```

5. Modulus
Mengembalikan sisa pembagian dari dua bilangan. Sangat berguna untuk mengecek bilangan ganjil atau genap.
```Python
# Modulus
x = 10
y = 3
z = x % y
print(x % y)
print(z)
print(10 % 3)
```

6. Perpangkatan
Menaikkan angka di sisi kiri ke pangkat angka di sisi kanan (Eksponen).
```Python
# Perpangkatan
x = 2 ** 3
print(x)
print(2 ** 3)
```

7. Pembagian Bulat
Membagi dua bilangan dan membulatkan hasilnya ke bawah ke bilangan bulat terdekat (Floor Division).
```Python
# Pembagian Bulat
x = 10
y = 3
z = x // y
print(x // y)
print(z)
print(10 // 3)
```
